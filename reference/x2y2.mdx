# Getting Started with X2Y2 on Reservoir

Reservoir offers access to X2Y2 & 150+ other marketplaces through our simple, and powerful APIs, SDKs & UI kits. To get started you can [get an API key](/reference/dashboard-sign-up) which has a generous [free plan & affordable paid subscriptions](https://reservoir.tools/pricing). X2Y2 orders and data are included in all of our endpoints. However, should you want to access X2Y2 data and trading specifically, you can. The following sections outline how to access X2Y2 data and trading using Reservoir.

# Getting X2Y2 Data & Orders

* **Sales** - Get X2Y2 Sales from our [Sales API](/reference/getsalesv5). Filtering using the API is not currently supported, so get all sales for a token, collection, or attribute, and filter on the front end using `"orderSource": "x2y2.io"`or `"fillSource": "x2y2.io"`. When using `orderSource` you are filtering by sales whose orders are native to X2Y2, while filtering on `fillSource` will return all sales that were filled on X2Y2, regardless of the order origination marketplace.

* **Asks (Listings)** - X2Y2 asks (listings) can be accessed through our [Asks API.](/reference/getordersasksv4) To get X2Y2 listings, ensure to set the `source=x2y2.io`. [An example of the response.](https://api.reservoir.tools/orders/asks/v4?source=x2y2.io)

* **Bids (Offers)** - X2Y2 bids (offers) can be accessed through our [Bids API.](/reference/getordersbidsv5) To get X2Y2 Offers, ensure to set the `source=x2y2.io`. [An example of the response.](https://api.reservoir.tools/orders/bids/v5?source=x2y2.io)

# Trading on X2Y2 through Reservoir

* **Buying** - Get the [listings data](/reference/getordersasksv4) and fill those orders either using the [API ](/reference/postexecutebuyv7)or [SDK](/reference/buytoken) *(recommended)*
* **Selling** - Get the [bid data](/reference/getordersbidsv5) and fill the bids either using the [API ](/reference/postexecutesellv7)or [SDK](/reference/acceptoffer) *(recommended)*
* **Creating Asks (Listings)** - Create the listing through the [API ](/reference/postexecutelistv5)or the [SDK](/reference/listtoken) *(recommended)*, set the `"orderKind": "x2y2"` & `"orderbook": "x2y2"` and [check the cross-post status](/reference/getcrosspostingordersv1)
* **Creating Bids (Offers)** - Create the offer through the [API ](/reference/postexecutebidv5)or the [SDK](/reference/placebid) *(recommended)* set the `"orderKind": "x2y2"` & `"orderbook": "x2y2"` and [check the cross-post status](/reference/getcrosspostingordersv1).
