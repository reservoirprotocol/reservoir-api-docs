# Overview

The Reservoir API is an all-in-one endpoint for building NFT applications. Get started for free by creating an account on our [Developer Dashboard](https://auth.reservoir.tools/u/login?state=hKFo2SBJUmJTUXhzc1lpVVpEdDZaZm5sZnRfdnRvc0NiRmlUTqFur3VuaXZlcnNhbC1sb2dpbqN0aWTZIDVGQktHcFJDV3V0UnhfYzl1ZXBaWjJKZmNSNG0wQk9Do2NpZNkgN3BPWXNyNDE5c0ROUnB3Qm13NE5KZXRnNWQ4RUhWWEU).

## NFT trading

* [Buy tokens](/reference/postexecutebuyv7) and [accept bids](/reference/postexecutesellv7) from major marketplaces
* Create your own [orderbook](/reference/postorderv4), including advanced order types
* Cross post [bids](/reference/postexecutebidv5) and [asks](/reference/postexecutelistv5) to other major marketplaces

Learn more about our NFT trading APIs [here](/reference/creating-and-filling-orders).

## NFT data

* [Granular token price data](/reference/gettokensfloorv1) with flexible querying options
* Real-time [collection floor price](/reference/geteventscollectionsflooraskv2) and [top bid](/reference/geteventscollectionstopbidv2) events
* Aggregated NFT [orders](/reference/orders), for executing buys and sells
* Detailed [metadata](/reference/getcollectionsv5) that can be used to analyze data (e.g. [Collection attributes](/reference/attributes))
* Token [ownership data](/reference/owners) and [collection statistics](/reference/getstatsv2)
* Leverage NFT price and token status oracles
* Get real-time and historical [royalty data](/docs/royalties)

Learn more about all our NFT data endpoints [here](/reference/nft-data-overview).

# Versioning

In order to ensure backwards compatibility, any changes to API endpoints are deployed as new versions. We will strive to keep old versions available for as long as possible, while also working closely with you to help upgrade.
