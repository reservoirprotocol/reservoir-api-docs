# Getting Started with Redstone on Reservoir

Reservoir offers access to Redstone & a number of EVM-compatible chains through our simple, yet powerful APIs, SDKs & UI kits for NFT trading and data. To get started [create an account](https://dashboard.reservoir.tools/) and [get an API key](/reference/dashboard-sign-up).

* **Redstone API Base URL** - You can access all Reservoir APIs by using the `https://api-redstone.reservoir.tools` base URL. When using the API reference docs for Redstone, make sure to change the base URL to the one above.
* **Recommended Test Collection** - Our recommended Redstone NFT collection to test with is [Reservoir ERC721 Test Collection](https://explorer.redstone.xyz/address/0xB00FBF022bd3Fc69B7EC2DE063a673dbDe931450) contract address: `0xb00fbf022bd3fc69b7ec2de063a673dbde931450`. When using the API reference docs, use the collection above to test responses.

To get a better sense of what we offer, check out of reference [Redstone Marketplace & Explorer](https://explorer.reservoir.tools/redstone).

## Trading on Redstone with Reservoir

Reservoir makes it simple to not only get market data and orders, but execute and create orders directly through our APIs and tools. To learn more about trading with Reservoir, check out our [NFT Trading Overview](/reference/creating-and-filling-orders). On Redstone, you can:

**Execute orders** from across major marketplaces, by using the trading APIs or the SDK, which wraps the APIs into simple methods. Check out the buy and sell endpoints below:

* [Buy Redstone NFTs](/reference/postexecutebuyv7)
* [Sell Redstone NFTs](/reference/postexecutesellv7)

**Create orders** with [custom fees](/docs/custom-fees) and [royalties](/docs/royalties) that will get distributed across the Reservoir ecosystem. Or cross-post orders to major marketplaces that have Redstone support. Check out our list and bid endpoints below:

* [List Redstone NFTs](/reference/postexecutelistv5)
* [Bid on Redstone NFTs](/reference/postexecutebidv5)
