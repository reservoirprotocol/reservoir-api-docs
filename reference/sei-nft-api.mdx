# Getting Started with Sei on Reservoir

Reservoir offers access to Sei & a number of EVM-compatible chains through our simple, yet powerful APIs, SDKs & UI kits for NFT trading and data. To get started [create an account](https://dashboard.reservoir.tools/) and [get an API key](/reference/dashboard-sign-up).

* **Sei API Base URL** - You can access all Reservoir APIs by using the `https://api-sei.reservoir.tools` base URL. When using the API reference docs for Sei, make sure to change the base URL to the one above.
* **Recommended Test Collection** - Our recommended Sei NFT collection to test with is [Reservoir ERC721 Test Collection](https://seitrace.com/address/0x614Fbad91b77cf487F6782DEAab9da057D41e7EE?chain=atlantic-2) contract address: `0x614Fbad91b77cf487F6782DEAab9da057D41e7EE `. When using the API reference docs, use the collection above to test responses.

To get a better sense of what we offer, check out of reference [Sei Marketplace & Explorer](https://explorer.reservoir.tools/sei).

## Trading on Sei with Reservoir

Reservoir makes it simple to not only get market data and orders, but execute and create orders directly through our APIs and tools. To learn more about trading with Reservoir, check out our [NFT Trading Overview](/reference/creating-and-filling-orders). On Sei, you can:

**Execute orders** from across major marketplaces, by using the trading APIs or the SDK, which wraps the APIs into simple methods. Check out the buy and sell endpoints below:

* [Buy Sei NFTs](/reference/postexecutebuyv7)
* [Sell Sei NFTs](/reference/postexecutesellv7)

**Create orders** with [custom fees](/docs/custom-fees) and [royalties](/docs/royalties) that will get distributed across the Reservoir ecosystem. Or cross-post orders to major marketplaces that have Sei support. Check out our list and bid endpoints below:

* [List Sei NFTs](/reference/postexecutelistv5)
* [Bid on Sei NFTs](/reference/postexecutebidv5)
