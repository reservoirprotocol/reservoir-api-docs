# Getting Started with Nebula on Reservoir

Reservoir offers access to Nebula & a number of EVM-compatible chains through our simple, yet powerful APIs, SDKs & UI kits for NFT trading and data. To get started [create an account](https://dashboard.reservoir.tools/) and [get an API key](/reference/dashboard-sign-up).

* **Nebula API Base URL** - You can access all Reservoir APIs by using the `https://api-nebula.reservoir.tools` base URL. When using the API reference docs for Nebula, make sure to change the base URL to the one above.
* **Recommended Test Collection** - Our recommended Nebula NFT collection to test with is [Reservoir ERC721 Test Collection](https://green-giddy-denebola.explorer.mainnet.skalenodes.com/address/0xE2537569B2F5C320Db0C5b2510728d8de0da28e0) contract address: `0xe2537569b2f5c320db0c5b2510728d8de0da28e0`. When using the API reference docs, use the collection above to test responses.

To get a better sense of what we offer, check out of reference [Nebula Marketplace & Explorer](https://explorer.reservoir.tools/nebula).

## Trading on Nebula with Reservoir

Reservoir makes it simple to not only get market data and orders, but execute and create orders directly through our APIs and tools. To learn more about trading with Reservoir, check out our [NFT Trading Overview](/reference/creating-and-filling-orders). On Nebula, you can:

**Execute orders** from across major marketplaces, by using the trading APIs or the SDK, which wraps the APIs into simple methods. Check out the buy and sell endpoints below:

* [Buy Nebula NFTs](/reference/postexecutebuyv7)
* [Sell Nebula NFTs](/reference/postexecutesellv7)

**Create orders** with [custom fees](/docs/custom-fees) and [royalties](/docs/royalties) that will get distributed across the Reservoir ecosystem. Or cross-post orders to major marketplaces that have Nebula support. Check out our list and bid endpoints below:

* [List Nebula NFTs](/reference/postexecutelistv5)
* [Bid on Nebula NFTs](/reference/postexecutebidv5)
