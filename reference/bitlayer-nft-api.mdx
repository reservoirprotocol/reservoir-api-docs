# Getting Started with Bitlayer on Reservoir

Reservoir offers access to Bitlayer & a number of EVM-compatible chains through our simple, yet powerful APIs, SDKs & UI kits for NFT trading and data. To get started [create an account](https://dashboard.reservoir.tools/) and [get an API key](/reference/dashboard-sign-up).

* **Bitlayer API Base URL** - You can access all Reservoir APIs by using the `https://api-bitlayer.reservoir.tools` base URL. When using the API reference docs for Bitlayer, make sure to change the base URL to the one above.
* **Recommended Test Collection** - Our recommended Bitlayer NFT collection to test with is [Reservoir ERC721 Test Collection](https://www.btrscan.com/address/0xb4f5037a20787984c07e05fd5a7d2c5ec3adec3c?tab=Transactions) contract address: `0xb4f5037a20787984c07e05fd5a7d2c5ec3adec3c`. When using the API reference docs, use the collection above to test responses.

To get a better sense of what we offer, check out of reference [Bitlayer Marketplace & Explorer](https://explorer.reservoir.tools/bitlayer).

## Trading on Bitlayer with Reservoir

Reservoir makes it simple to not only get market data and orders, but execute and create orders directly through our APIs and tools. To learn more about trading with Reservoir, check out our [NFT Trading Overview](/reference/creating-and-filling-orders). On Bitlayer, you can:

**Execute orders** from across major marketplaces, by using the trading APIs or the SDK, which wraps the APIs into simple methods. Check out the buy and sell endpoints below:

* [Buy Bitlayer NFTs](/reference/postexecutebuyv7)
* [Sell Bitlayer NFTs](/reference/postexecutesellv7)

**Create orders** with [custom fees](/docs/custom-fees) and [royalties](/docs/royalties) that will get distributed across the Reservoir ecosystem. Or cross-post orders to major marketplaces that have Bitlayer support. Check out our list and bid endpoints below:

* [List Bitlayer NFTs](/reference/postexecutelistv5)
* [Bid on Bitlayer NFTs](/reference/postexecutebidv5)
