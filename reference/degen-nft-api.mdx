# Getting Started with Degen on Reservoir

Reservoir offers access to Degen & a number of EVM-compatible chains through our simple, yet powerful APIs, SDKs & UI kits for NFT trading and data. To get started [create an account](https://dashboard.reservoir.tools/) and [get an API key](/reference/dashboard-sign-up).

* **Degen API Base URL** - You can access all Reservoir APIs by using the `https://api-degen.reservoir.tools` base URL. When using the API reference docs for Degen, make sure to change the base URL to the one above.
* **Recommended Test Collection** - Our recommended Degen NFT collection to test with is [Reservoir ERC721 Test Collection](https://explorer.degen.tips/address/0xec28e501d65781966bcfbbec73555461b66cfb05) contract address: `0xec28e501d65781966bcfbbec73555461b66cfb05`. When using the API reference docs, use the collection above to test responses.

To get a better sense of what we offer, check out of reference [Degen Marketplace & Explorer](https://explorer.reservoir.tools/degen).

## Trading on Degen with Reservoir

Reservoir makes it simple to not only get market data and orders, but execute and create orders directly through our APIs and tools. To learn more about trading with Reservoir, check out our [NFT Trading Overview](/reference/creating-and-filling-orders). On Degen, you can:

**Execute orders** from across major marketplaces, by using the trading APIs or the SDK, which wraps the APIs into simple methods. Check out the buy and sell endpoints below:

* [Buy Degen NFTs](/reference/postexecutebuyv7)
* [Sell Degen NFTs](/reference/postexecutesellv7)

**Create orders** with [custom fees](/docs/custom-fees) and [royalties](/docs/royalties) that will get distributed across the Reservoir ecosystem. Or cross-post orders to major marketplaces that have Degen support. Check out our list and bid endpoints below:

* [List Degen NFTs](/reference/postexecutelistv5)
* [Bid on Degen NFTs](/reference/postexecutebidv5)
