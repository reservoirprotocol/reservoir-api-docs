# Getting Started with Minato on Reservoir

Reservoir offers access to Minato -- Soniuem's official testnet -- through our simple, yet powerful APIs, SDKs & UI kits for NFT trading and data. To get started [create an account](https://dashboard.reservoir.tools/) and [get an API key](/reference/dashboard-sign-up).

* **Minato API Base URL** - You can access all Reservoir APIs by using the `https://api-minato.reservoir.tools` base URL. When using the API reference docs for Minato, make sure to change the base URL to the one above.
* **Recommended Test Collection** - Our recommended Minato NFT collection to test with is [Reservoir ERC721 Test Collection](https://explorer-testnet.soneium.org/address/0x19470C329721255B3586906E956855d868c80147) contract address: `0x19470c329721255b3586906e956855d868c80147 `. When using the API reference docs, use the collection above to test responses.

To get a better sense of what we offer, check out of reference [Minato Marketplace & Explorer](https://testnets.reservoir.tools/minato).

## Trading on Minato with Reservoir

Reservoir makes it simple to not only get market data and orders, but execute and create orders directly through our APIs and tools. To learn more about trading with Reservoir, check out our [NFT Trading Overview](/reference/creating-and-filling-orders). On Minato, you can:

**Execute orders** from across major marketplaces, by using the trading APIs or the SDK, which wraps the APIs into simple methods. Check out the buy and sell endpoints below:

* [Buy Minato NFTs](/reference/postexecutebuyv7)
* [Sell Minato NFTs](/reference/postexecutesellv7)

**Create orders** with [custom fees](/docs/custom-fees) and [royalties](/docs/royalties) that will get distributed across the Reservoir ecosystem. Or cross-post orders to major marketplaces that have Minato support. Check out our list and bid endpoints below:

* [List Minato NFTs](/reference/postexecutelistv5)
* [Bid on Minato NFTs](/reference/postexecutebidv5)
