# Getting Started with Xai on Reservoir

Reservoir offers access to Xai & a number of EVM-compatible chains through our simple, yet powerful APIs, SDKs & UI kits for NFT trading and data. To get started [create an account](https://dashboard.reservoir.tools/) and [get an API key](/reference/dashboard-sign-up).

* **Xai API Base URL** - You can access all Reservoir APIs by using the `https://api-xai.reservoir.tools` base URL. When using the API reference docs for Xai, make sure to change the base URL to the one above.
* **Recommended Test Collection** - Our recommended Xai NFT collection to test with is [Reservoir ERC721 Test Collection](https://explorer.xai-chain.net/address/0x9484A31F51108F095fcd7Aa2bB8d22340d20C422) contract address: `0x9484A31F51108F095fcd7Aa2bB8d22340d20C422`. When using the API reference docs, use the collection above to test responses.

To get a better sense of what we offer, check out of reference [Xai Marketplace & Explorer](https://explorer.reservoir.tools/xai).

## Trading on Xai with Reservoir

Reservoir makes it simple to not only get market data and orders, but execute and create orders directly through our APIs and tools. To learn more about trading with Reservoir, check out our [NFT Trading Overview](/reference/creating-and-filling-orders). On Xai, you can:

**Execute orders** from across major marketplaces, by using the trading APIs or the SDK, which wraps the APIs into simple methods. Check out the buy and sell endpoints below:

* [Buy Xai NFTs](/reference/postexecutebuyv7)
* [Sell Xai NFTs](/reference/postexecutesellv7)

**Create orders** with [custom fees](/docs/custom-fees) and [royalties](/docs/royalties) that will get distributed across the Reservoir ecosystem. Or cross-post orders to major marketplaces that have Xai support. Check out our list and bid endpoints below:

* [List Xai NFTs](/reference/postexecutelistv5)
* [Bid on Xai NFTs](/reference/postexecutebidv5)
