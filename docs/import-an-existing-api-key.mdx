title: "Import an Existing API Key"

---

Any existing and unclaimed API key can be easily imported into your companyâ€™s [Reservoir account](/reference/rate-limits) with the existing rate limits remaining the same at no additional cost.

To import your existing key, simply navigate to the App page and click the Import Key button.

![](/images/docs/28744ed-small-Screen_Shot_2023-04-26_at_3.20.18_PM.png)

Enter the required details and click Import.

![](/images/docs/cc32433-small-Screen_Shot_2023-04-26_at_3.20.12_PM.png)

An email will be sent to the original creator of the key. Open this email and accept the request to import the key.

Note: If you no longer have access to the email that created the key please [contact our support team](/docs#contact-us) for assistance.
